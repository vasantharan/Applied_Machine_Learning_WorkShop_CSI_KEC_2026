## What is Docker (Containerization)

Docker packages application + dependencies into one container.

**Why Docker?**
- Works same everywhere
- No dependency issues
- Easy production deployment
- Scalable
- Industry standard

**Key Concepts:**
- Dockerfile
- Image
- Container
- Port mapping

**Benefits for ML:**
- Package model + API together
- Avoid version conflicts
- Easy cloud migration
- Production-ready system
